#MVVM-Architecture-PhotoApp

Overview :

The app was written in Kotlin.
REST photo API: Get information about countries via a RESTful API .
The API’s Docs: https://www.pexels.com/

Technologies and Libraries Used :

<li>MVVM</li>
<li>Retrofit</li>
<li>Paging 3</li>
<li>Dagger Hilt</li>
<li>Glide</li>
